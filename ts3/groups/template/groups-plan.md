## Estratégia

1. criar grupos por nome
2. capturar o SGID novo criado
3. aplicar permissões equivalentes
4. manter ordem lógica por bloco
5. deixar futuros grupos de level expansíveis

## Observação

Os SGIDs da instância atual são referência apenas de origem.
Na nova instância, o script deve trabalhar com os IDs gerados localmente.
