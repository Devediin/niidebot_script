# Group Blocks

Resumo lógico dos grupos TS3 para o template.

## Bloco 1 - Staff/Base

- Invisível
- Leader
- Server Admin
- Movedor
- Registrada
- Registrado
- Sem Descricao
- Spy

### Observação
`Spy` é grupo padrão/nativo do TS3 e não deve ser recriado como grupo custom.

## Bloco 2 - NiideBot

- Bot Admin
- Bot Moderator

## Bloco 3 - Vocações

- EK
- ED
- RP
- MS
- EM

Observação:
as permissões das vocações serão tratadas com base comum.

## Bloco 4 - Level

- `-- LEVEL --`
- grupos de level

Observação:
ainda não foi detalhado por completo, mas a estratégia será usar base comum quando aplicável.

## Regra geral

Na nova instância:
- recriar grupos custom
- aplicar permissões equivalentes
- manter ordem lógica

No servidor atual:
- não apagar
- não recriar
- não mexer nos IDs existentes
