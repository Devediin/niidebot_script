# TS3 Template

Esta pasta contém os arquivos relacionados ao template do TeamSpeak 3.

## Objetivo

Permitir recriar em novas instâncias a mesma estrutura usada no ambiente atual:

- grupos
- permissões
- canais
- ordem lógica
